# HeinoDiscord Plugin Registry

HeinoDiscord uses a source-based plugin model. The safest "cloud" is a public
Git repository plus a static registry file, not remote code that silently runs
on users' computers.

Recommended hosting:

1. Push this repository to GitHub.
2. Enable GitHub Pages or use raw GitHub URLs for `opencord/cloud/registry.json`.
3. Review plugin pull requests in public.
4. Build releases from source.

This keeps the project 100% open source and avoids hidden monetized or
obfuscated plugin delivery.
